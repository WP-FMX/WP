A better WP experience

(an illustrated guide)

The WrongPlanet website has a few "quirks" that I find annoying. While we, the users, cannot change what the server does there are some things we can do in our browsers to minimise the annoyances.

Below is a simple guide to:

  1. Remove ads and other visual clutter
  2. Make hyperlinks look "normal" (blue if unvisited, purple if visited)
  3. Allow post text to be recovered if your browser crashes

Note the difference:

This guide works in Google Chrome and Mozilla Firefox. If you're still using Internet Explorer - go and download Chrome from https://www.google.com/chrome
