Instructions for Chrome

Step 1: Install extensions

Follow each of these links and click ADD TO CHROME

  • AdBlock Plus: https://chrome.google.com/webstore/detail/adblock-plus/cfhdojbkjhnklbpkdaibdccddilifddb
  • Stylish: https://chrome.google.com/webstore/detail/stylish/fjnbnpbmkenffdnngjfgmeleoegfcffe
  • Lazarus: https://chrome.google.com/webstore/detail/lazarus-form-recovery/loljledaigphbcpfhfmgopdkppkifgno

Step 2: Configure extensions

Open extension settings from the menu (Tools, Extensions):

AdBlock Plus

The default filter list (EasyList) is enough to block the ads, but add a custom one to remove more clutter:

  1. Click Options under AdBlock Plus
  2. Click + Add filter subscription.
  3. Choose Add a different subscription...
  4. Under Filter list location enter https://raw.github.com/WP-FMX/WP/master/filter.txt (you can leave "Subscription title" blank)
  5. Click + Add
  6. Close the AdBlock Plus options tab and the Extensions tab

Stylish

Go to http://userstyles.org/styles/90807/wrongplanet-forum-hyperlink-fix and click + Install with Stylish
